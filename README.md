# ⚡ angel-script-ai

Fast and local AI model built to generate and optimize scripts directly on my PC. Powered by Ollama, running 100% offline.

### Key Features:
- **Zero Filler:** Direct functional code without intros or greetings.
- **100% Local:** Private and offline with no API keys.
- **Script-Focused:** Designed for game dev and logic.
Markdown
## How to Install & Run

1. Clone this repository:
git clone [https://github.com/I-No-BigM-Script/angel-script-ai.git](https://github.com/I-No-BigM-Script/angel-script-ai.git)
cd angel-script-ai
Create and run the AI model with Ollama:

Bash
ollama create angel-script-ai -f ./Modelfile
ollama run angel-script-ai
